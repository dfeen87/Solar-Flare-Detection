## Data Sources

**NOAA Space Weather Prediction Center (SWPC) — GOES JSON Data Feeds**  
National Oceanic and Atmospheric Administration.  
URL: <https://services.swpc.noaa.gov/json/goes/primary/>  
Accessed via public API endpoints:
- `xrays-7-day.json` — GOES X-ray flux (0.1–0.8 nm and 0.05–0.4 nm channels)
- `xray-flares-7-day.json` — Flare event catalogue with class and timing
- `xray-background-7-day.json` — Quiet-Sun X-ray background levels
- `magnetometers-7-day.json` — GOES magnetometer field measurements
- `euvs-7-day.json` — Extreme ultraviolet spectral irradiance (EUVS)
